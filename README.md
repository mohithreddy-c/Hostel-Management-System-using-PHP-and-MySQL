# Hostel Management System using PHP & MySQL

A web-based Hostel Management System developed using **PHP**, **MySQL**, **HTML**, **CSS**, and **JavaScript**.  
This project helps manage hostel operations such as student registration, room allocation, booking, and activity logs through a simple and user-friendly interface.

>  This project is intended for **learning and demonstration purposes only**.
